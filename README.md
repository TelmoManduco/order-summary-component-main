A simple and clean order summary card built with HTML and CSS. The layout is fully responsive and includes:
- A hero image
- Plan details with price
- Change plan link
- Payment button
- Cancel order link

- Flexbox
- Mobile-first workflow
- Google Fonts (Red Hat Display)
